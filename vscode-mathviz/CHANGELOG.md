# Changelog

## [0.1.0] - 2026-02-10

### Added
- Initial release
- Full TextMate grammar for MathViz DSL syntax highlighting
- Support for `.mviz` and `.mvz` file extensions
- Language configuration (brackets, comments, folding, indentation)
- 18 code snippets for common MathViz patterns
- Manim class, function, and constant recognition
- Unicode math operator and constant support
- String interpolation highlighting
- Decorator support (`@jit`, `@njit`, `@vectorize`, `@parallel`)
- Tokyo Night theme color customizations
